# User Stories - Customer Accounts Microservice

## Sprint 1: Foundation
1. **As a** developer, **I want to** set up the development environment **so that** I can start building the application.
2. **As a** developer, **I want to** create a RESTful API **so that** I can manage customer accounts.
3. **As a** developer, **I want to** implement CRUD operations **so that** I can create, read, update, and delete accounts.

## Sprint 2: Testing & CI
4. **As a** developer, **I want to** write unit tests **so that** I can ensure code quality.
5. **As a** developer, **I want to** set up GitHub Actions **so that** I can automate continuous integration checks.

## Sprint 3: Security & Deployment
6. **As a** developer, **I want to** add security headers **so that** the API is protected.
7. **As a** developer, **I want to** containerize the application with Docker **so that** it can be deployed consistently.
8. **As a** developer, **I want to** deploy to Kubernetes **so that** the application is scalable.
9. **As a** developer, **I want to** create a CD pipeline **so that** deployment is automated.
